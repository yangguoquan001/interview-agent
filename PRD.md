# 产品需求文档

## 产品概述

**产品名称**: AI 模拟面试 Agent
**产品定位**: 基于知识库的 AI 模拟面试系统，帮助用户练习技术面试

## 功能需求

### 1. 面试流程
- [x] 用户点击"开始面试"启动面试
- [x] 系统从知识库读取文档生成面试题
- [x] 用户输入回答，系统进行评估
- [x] 面试官根据回答进行追问
- [x] 用户可以主动结束面试

### 2. 历史记录管理
- [x] 左侧边栏显示按日期分组的历史记录
- [x] 点击记录可查看详细内容
- [x] 面试结束后自动保存记录到 records 目录

### 3. 面试题生成
- [x] 基于知识库文档内容生成面试题
- [x] 支持不同难度级别 (Easy/Medium/Hard)
- [x] 题目包含知识点来源和主题

## 数据模型

### AgentState
- `files_to_read`: 待读取的知识库文件列表
- `messages`: 对话历史
- `current_file`: 当前使用的文件
- `question`: 生成的面试题
- `user_answer`: 用户回答
- `feedback`: 评估反馈
- `topic`: 面试主题
- `difficulty`: 难度级别
- `is_end`: 是否结束面试
- `next`: 下一个要执行的节点

### InterviewRecord
- `file_path`: 文件路径
- `date`: 日期
- `topic`: 主题
- `content`: 内容
