# Interview Agent - 产品需求文档

## 1. 产品概述

### 1.1 产品名称
Interview Agent

### 1.2 产品类型
AI 模拟面试工具

### 1.3 核心价值
通过自动化扫描本地知识库，生成个性化的面试问题，帮助用户进行技术面试准备。支持回答评估、反馈改进和深度追问功能。

### 1.4 目标用户
- 正在准备技术面试的求职者
- 需要练习特定技术领域的开发者
- 希望巩固技术知识的学习者

---

## 2. 功能需求

### 2.1 知识库管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 目录扫描 | 自动扫描配置的目录下的所有 `.md` 文件 | P0 |
| 文件过滤 | 支持排除特定目录和文件 (如 README.md) | P0 |
| 随机选择 | 从候选文件中随机选取作为面试素材 | P0 |

### 2.2 面试出题

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 问题生成 | 基于文档内容生成面试题 | P0 |
| 难度分级 | 支持 Easy/Medium/Hard 三种难度 | P0 |
| 主题提取 | 自动识别并记录面试主题 | P0 |

### 2.3 回答评估

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 评分反馈 | 基于参考文档给出评分 (0-10) | P0 |
| 改进建议 | 提供具体可执行的改进建议 | P0 |
| 标准答案 | 提供参考的满分答案示例 | P0 |

### 2.4 追问答疑

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 技术追问 | 支持用户针对反馈进行深入提问 | P0 |
| 底层原理 | 回答需包含底层原理和最佳实践 | P0 |
| 多轮对话 | 支持多轮追问，不设限 | P1 |

### 2.5 进度管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 状态保存 | SQLite 自动保存面试进度 | P0 |
| 进度恢复 | 支持中断后从断点继续 | P0 |
| 记录导出 | 面试记录保存为 Markdown 文件 | P0 |

---

## 3. 非功能需求

### 3.1 性能要求
- 单次面试题生成响应时间 < 30 秒
- 评估反馈响应时间 < 20 秒

### 3.2 可用性要求
- 支持 CLI 交互式界面 (main.py)
- 支持 Streamlit Web 界面 (web_app.py)
- 清晰的输入输出提示

### 3.3 可扩展性
- 预留 Streamlit Web UI 接口
- 支持自定义提示词模板
- 支持多端入口 (CLI/Web)

## 4. 用户交互流程

### 4.1 CLI 模式 (main.py)

```
1. 启动程序
      │
      ▼
2. 扫描知识库 ──▶ 随机选择文件
      │
      ▼
3. 生成面试题 (显示难度、主题)
      │
      ▼
4. 用户输入回答
      │
      ▼
5. AI 评估反馈 (评分 + 建议 + 标准答案)
      │
      ▼
6. 用户选择: 追问 或 下一题
      │
      ├─ 追问 ──▶ chat_node ──▶ 回到步骤 6
      │
      └─ 下一题 ──▶ 保存记录 ──▶ 回到步骤 3
      │
      ▼
7. 所有题目完成，退出
```

### 4.2 Web 模式 (web_app.py)

```
1. 打开 Web 页面
      │
      ▼
2. Sidebar 显示历史记录 (按日期分组)
      │
      ▼
3. 选择"开始新面试"或查看历史记录
      │
      ├─ 查看历史 ──▶ 显示 .md 内容
      │
      └─ 开始面试 ──▶ 聊天窗口
           │
           ▼
      生成面试题 (流式输出)
           │
           ▼
      用户输入回答
           │
           ▼
      AI 评估反馈 (流式输出)
           │
           ▼
      用户选择: 追问 或 下一题
           │
           ├─ 追问 ──▶ 继续对话
           │
           └─ 下一题 ──▶ 保存记录 → 下一题
```
